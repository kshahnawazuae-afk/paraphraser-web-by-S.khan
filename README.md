# AI Text Paraphraser (Web Edition)

A free online tool to paraphrase text using multi-hop back-translation.

## Features
- Spanish and French translation paths
- Adjustable stealth level (Quality to Max Stealth)
- Sentence-by-sentence or all-at-once processing
- AI watermark removal
- Works on phone and desktop

## How to Use
1. Paste your text
2. Choose language and stealth level
3. Click "Rewrite Text"
4. Copy or download the result

## Deploy Your Own
This app runs on Streamlit Cloud. To deploy your own copy:
1. Fork this repository
2. Go to share.streamlit.io
3. Deploy from your repository
